# 🏪 店铺经营管理系统

一个简洁的店铺经营管理系统，支持记账、交易明细、数据分析等功能。

## ✨ 功能

- 📈 **仪表盘** — 收支概览、趋势图表、时间范围筛选
- ✏️ **记账** — 快速记录收入/支出，支持自定义分类
- 📋 **交易明细** — 搜索、筛选、分页、编辑、删除
- 🔬 **数据分析** — 月度对比、分类排行、支付方式分布
- ⚙️ **设置** — 分类管理、数据导出 CSV

## 🛠️ 技术栈

- **后端**: Node.js + Express + better-sqlite3
- **前端**: 原生 HTML/CSS/JS + Chart.js
- **数据库**: SQLite
